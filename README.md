# 🐍 Snake Game (صُنعت بواسطة ذكاء Sandy تحت إشرافي)

لعبة الحيّة الكلاسيكية بأسلوب عصري وانسيابي — تحكّم بالحيّة، اجمع الحبات، وحقّق أعلى سكور، وانتبه من حواف الشاشة!

---

## 🤖 مشروع أوتوماتيكي بالكامل: كيف تعمل Sandy التي قمتُ ببرمجتها؟

هذا المشروع ليس مجرد لعبة، بل هو استعراض حي لقوة نظام **Sandy** — وهو نظام وكيل ذكي ومستقل (Autonomous Agent) **قمتُ أنا ببرمجته وتطوير بنيته بالكامل** ليعمل كمهندس برمجيات ينفذ خطط التطوير المعقدة.

كل ما فعلته هو إرسال رسالة نصية واحدة لـ **ساندي (Sandy)** عبر تلغرام:

> «ابني تطبيق ويب (لعبة سلم وحية) باستخدام HTML و CSS و js فقط. تطبيق بسيط، عدّاد فوق يعد كم حبة تم اكلها، ولو لمست الشاشة أي من حواف الشاشة تطلع جملة Game Over، يتصفّر العدّاد ونبدا من جديد. ريبو خارجي»

### ⚙️ كواليس العمل والتنفيذ القائم على المجموعات (Groups)
بمجرد استلام الأمر، قامت ساندي بتقسيم المهمة هندسياً وتحديثي خطوة بخطوة عبر نظام مجموعات منظم، حيث كانت تطلب موافقتي قبل الانتقال لكل مرحلة:

*   🔔 📦 **Group 1/3: الإعداد** (task `sa_projectbuilder_33de975a`)  
    قامت بتهيئة الإعدادات الأساسية وإنشاء ملف: `config/gameSettings.js` ثم انتطرت موافقتي للنقل للمرحلة التالية.
    
*   🔔 📦 **Group 2/3: النواة** (task `sa_projectbuilder_33de975a`)  
    قامت ببناء الهيكل الأساسي والمحرك الفعلي للعبة وإنشاء ملفات: `public/index.html`، `public/styles/main.css`، و `public/scripts/gameEngine.js`.
    
*   🔔 ✅ **المرحلة النهائية: إتمام البناء والـ CI**  
    خلصت بناء كل الـ groups ورفعت 5 ملفات كاملة على الـ branch الخاص بالمهمة: `sandy-task-sa_projectbuilder_33de975a` وانتظرت فحص الـ CI (Continuous Integration).

### 🔀 المراجعة والـ Merge البشري
بعد انتهاء ساندي من رفع الكود واجتياز كل الفحوصات الآلية بنجاح، قامت بفتح **Pull Request** رسمي، حيث قمتُ أنا بمراجعة الكود البرمجي بنفسي وعمل الـ **Merge** على الـ `main` branch ونشره سينمائياً على **GitHub Pages**!

## ✨ المميزات
- 🎮 **Pure Vanilla Tech:** مبنية باستخدام HTML5, CSS3, و JavaScript خالص — بدون أي مكتبات أو إطارات عمل خارجية.
- 🔢 **Live Counter:** عدّاد نقاط مباشر ودقيق يحسب عدد الحبات المأكولة فوراً.
- 💥 **Cinematic Game Over:** شاشة نهاية تفاعلية تظهر فوراً عند ملامسة الحواف بـ Smooth Feedback.
- 🔁 **Instant Reset:** إعادة تشغيل تلقائية وتصفير العداد لتبدأ مغامرة جديدة بلمحة بصر.
